About Latin Squares

A Latin square is an (n x n) grid of cells. Each cell contains exactly one symbol from the set {1, 2, ..., n} and every row and column contains each symbol exactly once. This app generates Latin squares uniformly at random using Jacobson and Matthews' Markov chain. If you'd like to know more about how their method works, I urge you to read their paper.

FAQ

Why, when I click to move to the next square, does it ofen not change?

To ensure that the Markov chain is ergodic, it is important that the underlying graph is not bipartite. Putting these loops in the algorithm ensure that the graph is non-bipartite.

How many proper squares do I pass through when I click to move?

You move from the square you are on to the first proper square found.

Is this Markov chain rapidly mixing?

Nobody knows, but everyone who I've every spoken to about it would guess that it was.
